  • GNU not Linux

    by librepup — gnulinuxfree software

    Factually, it’s GNU/Linux or GNU+Linux. GNU is the operating system and userland; Linux is the kernel. Calling the whole system just “Linux” strips the GNU Project of its deserved credit, falsely hands Torvalds all the glory, and ignores the fact that Linux is merely a kernel — not an operating system.
